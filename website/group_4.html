The three major programming control structures:
  • Sequence: The order in which the code statements are compiled (for example from left to right, line-by-line from the top of the file to the bottom).
  • Selection: Which parts of the code are compiled and which are ignored, for example in conditional statements where a certain path is compiled if a given condition is true, while the other path is ignored.
  • Iteration: Performing a set task a set number of times, for example in a loop where a block of code is to be repeated while a given condition is true.
Hypertext markup language (HTML) and markup languages:
HTML is a markup language, used to lay out the structure of text within a document. Items to be displayed on a webpage are enclosed in HTML tags which mark them as being headers, paragraphs, table rows and so on, so that they are displayed correctly in a web browser.
Cascading style sheets (CSS):
CSS defines the look of a webpage, and is used to add images and colour as well as arranging elements on the page to allow for easier navigation. CSS is used in conjunction with HTML to add styling to the elements of the website that are enclosed in HTML tags.
Syntax and uses of programming languages:
The syntax of a programming language is the specific keywords it uses to perform functions and the rules and regulations for writing statements, for example whether statements must end with a semi-colon and which types of brackets are used for what purposes. Programming languages are used to send instructions to computers, performing countless functions from performing calculations to playing music.
